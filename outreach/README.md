# Outreach

Ship-ready submission packages for press, local papers, and community channels.

Each package contains:
- **Mail draft** (subject + body, copy-paste to email client)
- **Article** (final text, already run through voice checks)
- **Photo instructions** (where to render + export + caption + credit)
- **Author box** (optional block)
- **Submission checklist** (what to verify before sending)

Packages are self-contained markdown files. Treat them as frozen artefacts: if a version is sent, don't edit the file, duplicate it into `-v2.md` for later iterations.

## Workflow

1. Verify recipient email via each publication's Impressum
2. Export the photo (see Foto section inside each package)
3. Paste mail body, attach article + photo
4. Check the package's Submission-Checklist
5. Send
6. Update the "Sent?" column in this README after sending

## Voice rules (apply to all outreach)

- No em-dashes in user-facing copy (periods, commas, or semicolons instead)
- Self-implicating, concrete, no judgment of other families
- Short sentences mixed with reflective ones
- Brand nouns: **Ronki**, **ronki.de**, **hallo@ronki.de**
- Voice reference: `website/src/components/UeberMich.tsx` (the quote block is the tonal anchor)
